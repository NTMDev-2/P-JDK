_Packages are coming soon in future releases_

_Support for `import` is coming soon too_

To use a package, create the `.txt` inside this directory (or inside another directory inside of this folder). Then, inside your main file, declare an `import` statement:
```
import <path_to_package>;
```
Everything inside the package must have valid syntax. 

When support for `import` and packages are added, expect the `default` and `protected` access modifiers to be actually "working".
